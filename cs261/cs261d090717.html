Integer Number Systems

Give the bit patterns for the following values. Use 4 bits before and 4 bits after the binary point. For simplicity, feel free to omit the binary point)

  1. 6.5 using BCD encoding
  2. 6.5 using 2's compliment
  3. -6.5 using 2's compliment

Solutions

  1. 0110.0101
  2. 0110.1000
  3. 1001.1000

Using C Operators to Perform More

Using just the operators &, !, ~, &&, !!, ^, and >>, can you create a C function that returns 1 if the integer is positive and 0 if the integer is negative?
